# React Task Management App

## Description
A simple role-based task management application built with React. Admins can create, assign, and reassign tasks via drag-and-drop. Users can view and complete their tasks. Data is persisted in localStorage.

## Setup
1. Ensure Node.js (v14+) and npm are installed.
2. Clone or download the project.
3. Run `npm install` to install dependencies.
4. Run `npm start` to start the development server (opens at http://localhost:3000).

## Default Credentials
- Admin: username "admin", password "admin123"
- User: username "user1", password "user123"

## Features
- Role-based login and access.
- Task creation and assignment (Admin).
- Drag-and-drop reassignment (Admin).
- Task completion (User).
- Data persistence via localStorage.

## Testing
- Test login for both roles.
- Create tasks, assign, complete, and reassign via drag-and-drop.
- Clear localStorage to reset data.
